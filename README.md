# Queueless System

A MERN stack project to manage queues digitally.

## 🚀 Features
- Token generation
- Queue management
- Frontend using React
- Backend using Node.js and Express

## 🛠 Tech Stack
- React
- Node.js
- Express
- MongoDB

## ▶️ How to Run

### Client
cd client  
npm install  
npm start  

### Server
cd server  
npm install  
npm start
